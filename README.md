# Vanilla-MTG-Card-Tooltips

## Usage:

`[mtg]Card name[/mtg]` 

Will create a Link with a mouseover Tooltip of the card.

## Installation:

- Create a `MTG` folder under `plugins`
- Put `class.mtg.plugin.php` and the `js` folder into the `MTG` folder
- Open the Vanilla dashboard and activate the `MTG` Plugin
